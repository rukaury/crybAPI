# About Cryb

Cryb online rental property is developed and supported by students from the University of Ottawa.

Cryb is an online rental property which improves how content-based applications are built, managed and deployed, making customers more agile, innovative and successful. Cryb provides a next rental bidding system and enterprise ready platform for renting properties over the internet. Combining a powerful application development environment with php, mySql and an api using MVC architecture, the Cryb Platform provide clear business value to some of the most recognizable customers and agencies accross Canada. Cryb is headquartered in Ottawa.

More information is available at [www.cryb.ca](http://cryb.ca).
